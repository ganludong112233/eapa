# eapa
#EAPA介绍
eapa全程Exception And Performance Analysis，是一款用于java进程无侵入性性能和异常分析的工具。
#EAPA使用场景
分析已经存在的老代码调用请求链
对线上疑难bug和性能问题进行快速定位和诊断
#支持开发框架
目前仅仅支持mysql性能分析和基于springmvc、springboot的项目分析
#EAPA安装
eapa工具包包含三个内容
agent.jar
install.sh (目前只支持tomcat的安装，可以自己修改shell脚本)
eapa.properties 配置文件
